# 리모(Rimo) — 에이전트 구현 스펙

온라인 패션의 가짜 리뷰·정보 불신을 해결하는 리뷰 기반 신뢰형 커머스 플랫폼.
디자인 시스템은 **Seed Design System (당근마켓)** 을 사용한다.

---

## 절대 원칙

이 원칙은 어떤 구현 결정보다 우선한다. 예외 없음.

1. **홈은 리뷰 피드다** — 상품 피드 아님. 리뷰 카드가 커머스 진입점.
2. **구매 인증 배지는 항상 리뷰어명 바로 옆 첫 번째** — 모든 노출 지점(카드, 상세, 그리드, 리스트)에서.
3. **체형 정보는 항상 노출** — 숨기거나 접지 않는다. 키·몸무게·구매 사이즈는 카드 헤더 두 번째 줄.
4. **채팅은 카드에서 바로** — 별도 depth 없이 카드 내 버튼에서 직접 진입.
5. **핏 평가는 슬라이더만** — 자유 텍스트 입력 없음. 슬라이더 값이 필터링 데이터로 활용됨.
6. **채팅이 Primary CTA, 구매가 Secondary** — 리뷰 상세의 하단 버튼 순서 고정.
7. **리뷰 작성 Step 1은 반드시 구매 인증** — 인증 없이 체형/핏 입력 불가.
8. **포인트 대시보드가 마이 페이지 최상단** — 프로필 바로 아래. 리뷰어 동기부여 최우선.

---

## 앱 구조

```
Bottom Navigation: 홈 | 탐색 | [리뷰 작성 CTA] | 마이

서브 스크린 (Bottom Nav 없음):
  리뷰 상세  ← 홈/탐색 카드 탭
  채팅       ← 리뷰 상세 채팅 버튼
```

---

## 스크린별 핵심 결정

### 홈 피드
- 체형 필터 Chip Row가 피드 최상단 고정 (내 체형 기본 선택)
- 리뷰 카드 구조: 아바타 + [리뷰어명 + 구매 인증 배지] / 체형 정보 / 착용 이미지 3장 / 리뷰 텍스트 / 상품 미니 카드 / [채팅 문의 | 좋아요 | 저장]
- 상품 미니 카드는 리뷰 텍스트 아래 보조 정보로만 존재

### 리뷰 상세
- 최상단: 구매 인증 배지 + 구매 날짜 (신뢰 신호 먼저)
- 체형 카드: 키 / 몸무게 / 평소 사이즈 / 구매 사이즈 (2×2 그리드)
- 핏 평가: 어깨 폭 / 상체 길이 / 허리 라인 / 소재 두께 — Slider readOnly로 시각화
- 하단 CTA (고정): 1순위 채팅 문의, 2순위 "↗ 외부 링크로 구매하기"
- "↗ 외부 링크" 문구 필수 — 앱 밖 이동 사전 예고

### 탐색
- 내 체형 필터 카드가 검색창보다 위에 위치 (탐색의 시작점)
- 카테고리 Chip은 체형 필터 아래 (보조 필터)
- 그리드 카드에도 체형 정보(키·몸무게) + 구매 인증 배지 항상 노출

### 리뷰 작성 (4단계)
- Step 1: 구매 인증 업로드 → Step 2: 체형 정보 → Step 3: 핏 슬라이더 → Step 4: 사진 + 텍스트
- Step 1 안내에 포인트 지급 조건 + 개인정보 자동 마스킹 동시 명시
- 핏 슬라이더: 항목별 1~5 정수값, 양끝 레이블 필수 (좁음↔넓음 등)

### 채팅
- 채팅창 상단에 상품 컨텍스트 바 고정 (스크롤해도 유지)
- 헤더 부제: "OO 리뷰어" — 리뷰어 정체성 명확화

### 마이
- 포인트 대시보드: 보유 포인트 + 이번달 예상 / 연결 구매 건수 / 전환율 (3-stat)
- 리뷰 리스트 아이템에 리뷰별 적립 포인트 노출 — 고품질 리뷰 인센티브
- 프로필 부제에 체형 정보 (키·몸무게) 통합

## Seed 미지원 — 커스텀 구현 필요

| 요소 | 사유 |
|------|------|
| 채팅 버블 | Seed에 Chat UI 없음 |
| 착용 이미지 Carousel | Seed에 Carousel 없음 — 외부 라이브러리 사용 |
| Step 진행 표시바 | Seed에 Step indicator 없음 |
| 파일 업로드 존 | Seed에 Dropzone 없음 |
| 포인트 Stat 카드 | Seed 컴포넌트 조합으로 구현 |
| 핏 Dot Meter (상세) | Slider readOnly로 대체 가능 |
| Bottom Navigation | Seed에 없음 — 커스텀 구현 |
